THQ宣布《英雄连：抵抗前线》正式发售

http://www.sina.com.cn 2007年09月26日 15:51  新浪游戏

　　国际重量级游戏发行商THQ公司本周二正式宣布，由Relic Entertainment公司开发的二战即时战略游戏新作《英雄连：抵抗前线(Company of Heroes Opposing Fronts)》已正式在北美地区上市销售。作为《英雄连》系列第二部作品，全新的图像引擎、全新的出场单位、全新的游戏内容以及更为精彩的多人游戏体验，将使这部新作延续前作的成功。

　　

《英雄连：抵抗前线》封面

　　《英雄连：抵抗前线》游戏特色

　　・全新出场势力：英国红色贝蕾帽部队和纳粹德国精英装甲军。

　　・全新游戏战役：英国第2军以及德国精英装甲军的全新战斗故事，将为你从不同视角描绘

二战西线战争。

　　・真实的战争、真实的战场：动态天气变化、增强的载具战术等新增内容将向你展现真实残酷的战场。

　　・全新的多人游戏：包括新增的出场阵营在内，玩家一共可以选择控制四方势力进行联机对战。

　　・完全兼容DirectX 10的图像技术――获得大幅增强的图像引擎，将通过更加华丽的光影效果和更加真实的地表细节全面展现真实的战场环境。
